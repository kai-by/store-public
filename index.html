Политика конфиденциальности

Дата последнего обновления: 16 февраля 2026 г.

Настоящая Политика конфиденциальности описывает, как мобильное приложение Kaiby Store (далее — «Приложение») собирает, использует и защищает данные пользователей.

1. Какие данные мы собираем

  • Номер телефона — предоставляется пользователем добровольно при оформлении заявки на товар. Хранится на устройстве пользователя и передаётся на сервер при отправке заявки.
  • Данные об использовании — обезличенная техническая информация (тип устройства, версия операционной системы, информация о сбоях приложения) может собираться автоматически для улучшения работы Приложения.

2. Как мы используем данные

  • Номер телефона используется исключительно для связи с пользователем по его заявке на товар.
  • Техническая информация используется для диагностики ошибок и улучшения качества Приложения.

3. Передача данных третьим лицам

Мы не продаём и не передаём персональные данные пользователей третьим лицам, за исключением случаев, предусмотренных законодательством.

4. Хранение данных

  • Номер телефона хранится локально на устройстве пользователя для удобства повторного использования.
  • Данные заявок хранятся на защищённом сервере.

5. Безопасность

Мы принимаем разумные технические и организационные меры для защиты данных пользователей от несанкционированного доступа, изменения или уничтожения.

6. Права пользователя

Пользователь может в любой момент удалить свои данные из Приложения, очистив данные приложения в настройках устройства или удалив Приложение. Для удаления данных с сервера свяжитесь с нами.

7. Изменения

Мы оставляем за собой право обновлять настоящую Политику. Актуальная версия всегда доступна по данной ссылке.
